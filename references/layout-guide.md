# Layout Guide — Trip Destination Card

## 卡片尺寸

- **幻灯片模式**：每张卡片 `100vw × 100vh`，填满视口
- **长图模式**：每张卡片 `max-width: 1200px`，自动高度，纵向堆叠

## 单酒店布局（默认）

数据面板（52%）+ 图片面板（42%），左右并排，垂直居中。

```
┌──────────────────────────────────────────────────────┐
│  左 52%                          │  右 42%           │
│  目的地名称（h1，衬线大字）        │                   │
│  ─────────────────────────────── │  主图（4:3）       │
│  交通 · 航班                     │                   │
│  天气 · 温度                     │  缩略图 × 2        │
│  ─────────────────────────────── │  （可选）          │
│  推荐酒店（h2）+ 详情             │                   │
│  ◆ 优点 / ◇ 缺点                 │                   │
└──────────────────────────────────────────────────────┘
```

## 双酒店布局（三栏）

当一个目的地有两个酒店推荐时，使用三栏布局。三栏在卡片内垂直居中，总宽度 100%。

```
┌──────────────────────────────────────────────────────────────────┐
│  左栏 1fr        │  中栏 1.6fr              │  右栏 1.6fr         │
│  目的地名称       │  酒店 A 名称 + 详情       │  酒店 B 名称 + 详情  │
│  交通（纵向）     │  ◆ 优点 / ◇ 缺点          │  ◆ 优点 / ◇ 缺点    │
│  天气（纵向）     │  ┌──────────────────┐    │  ┌────────────────┐ │
│                  │  │  酒店 A 图片      │    │  │  酒店 B 图片    │ │
│                  │  │  (3:2)           │    │  │  (3:2)         │ │
│                  │  └──────────────────┘    │  └────────────────┘ │
└──────────────────────────────────────────────────────────────────┘
```

## 图片处理

- 用户提供图片文件时，转换为 base64 嵌入（推荐，离线可用）
- 用户提供 URL 时，保留 URL + 添加 `onerror` 回退
- 单酒店卡片：主图 1 张（4:3）+ 可选缩略图 2 张
- 双酒店卡片：每店 1 张图（3:2），位于优缺点下方
- 无图片时使用 SVG 占位符

## 翻页与导航

- 横向翻页使用 CSS `scroll-snap-type: x mandatory`
- 每张卡片 `scroll-snap-align: center`
- 导航点（底部圆点）自动生成，高亮当前页
- ESC 键显示全部卡片的缩略索引

## 响应式

- 单酒店：≤768px 时图片移到信息下方
- 双酒店：≤768px 时三栏变为单栏纵向堆叠

## 导出

- 切换到长图模式后，使用 `html-to-image` 库的 `toBlob()` 导出 PNG
- 导出时自动隐藏导航 UI
- 文件名格式：`行程方案-YYYYMMDD.png`

## 字体规格

| 元素 | 字体 | 字号 |
|---|---|---|
| 目的地标题 h1 | Noto Serif SC 900 | clamp(38px, 5.5vw, 72px) |
| 副标题 | Noto Serif SC 400 | clamp(16px, 1.8vw, 22px) |
| 交通/天气 | JetBrains Mono | 15px |
| 标签（交通/天气） | Noto Sans SC 700 | 11px |
| 航班行 | JetBrains Mono | 14px |
| 酒店名称 h2 | Noto Serif SC 600 | clamp(18px, 2.2vw, 26px) |
| 酒店详情 | JetBrains Mono | 14px |
| 优点/缺点 | Noto Sans SC | 14px |
